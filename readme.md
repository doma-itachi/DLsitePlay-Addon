# DLsitePlay Addon
## 概要
PC / Android版Firefox上で動作する、DLsitePlayに新しい機能を追加するアドオンです。

## インストール
### PC
### Android

## 機能
### 実装済み
- 読書進捗の保存＆ページ復帰
- データのバックアップ＆復帰

### 実装したい
- ダークモード

## パッケージ化
``` python pack.py src ```を実行してください
