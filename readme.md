# DLsitePlay Addon
## 概要
PC / Android版Firefox上で動作する、DLsitePlayに新しい機能を追加するアドオンです。

## インストール
### PC
### Android
1. Beta版またはNightly版Firefoxをインストール(Beta版を推奨)
2. 以下のユーザーID、リストIDから拡張機能をインストールしてください  

詳しくは[こちら]()

## 機能
### 実装済み
- 読書進捗の保存＆ページ復帰
- データのバックアップ＆復帰

### 実装したい
- ダークモード
- モーダルダイアログ

## パッケージ化
``` python pack.py src ```を実行してください
